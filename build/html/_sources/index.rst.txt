Variáveis Suportadas
=====================

Aqui estão as variáveis suportadas na Linguagem L:

- `int`: Números inteiros.
- `float`: Números de ponto flutuante.
- `str`: Strings de texto.

Exemplos:
x = 10  # um inteiro
y = 3.14  # um float
nome = "Ariano"  # uma string
